
# NEURON  
A framework agent-based system recomendation.  

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/fsant0s/neuron.git
~~~

Go to the project directory  

~~~bash  
  cd neuron
~~~

Install dependencies  
Install uv package manager and python 3.12. 

~~~bash  
uv --sync
~~~

Start the server  

~~~bash  
npm run start
~~~
